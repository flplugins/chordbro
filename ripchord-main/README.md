# ChordBro Core

ChordBro by FL Plugins is a premium-feeling fork of Ripchord focused on fast MIDI chord triggering, modern voicings, curated factory content, and a cleaner commercial release workflow around the open-source core.

# Open Source

The plugin core remains GPL-licensed. Commercial website assets, premium preset packs, screenshots, and sales collateral should live outside the GPL core and are documented separately in the repository root.
